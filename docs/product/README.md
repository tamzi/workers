# Product Documentation

This directory holds product requirements and planning artifacts.

## Files

- `prd.md` - Product requirements and goals.
- `featureList.md` - Feature inventory and status.
- `roadmap.md` - High-level delivery plan.

## What to Capture

- The product vision and target users.
- Scope for the current milestone.
- Planned features and priority order.
- Success metrics and risks.
