# Agent Rules Index

> **Purpose:** Centralized rules and guidelines for AI agents working on the repo

## Overview

This directory contains specific rules for planning, implementation, review, testing, and
documentation. Use it as the source of truth when making changes.

- `documentationRules.md` - Documentation organization rules
